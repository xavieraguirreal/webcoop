# Prompts para generación de imágenes con IA

Usar siempre estos prompts (o variaciones) para mantener un estilo visual consistente en todo el sitio.

## Estilo general
- Fotografía editorial, realista, cálida
- Golden hour / luz natural
- Tono documental, no corporativo
- 16:9, 1920x1080px, JPG calidad 80-90%

## Notas
- Siempre pedir "documentary photography style" para mantener autenticidad
- Evitar looks corporativos o demasiado pulidos
- Preferir luz natural, golden hour
- Incluir manos trabajando cuando sea posible (humaniza)
- Si se genera para el sitio del Taller Solidario, usar el mismo estilo pero con actividades culturales/educativas en vez de productivas
